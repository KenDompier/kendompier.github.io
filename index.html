Projects

Random YouTube Generator

I wrote this Python program to compensate for YouTube's lack of a random video feature. It generates a string of 3-5 random characters and searches for it with the YouTube API. When the program recieves the first result, it returns the link and opens it with the user's default search browser. The program also has the ability to search for video by length and before a certain year. I also gave the program a cute little face that reacts to errors!

Link

Textbook Twitter Bot

My first Twitter bot using Python! From a list of a few different textbook publishers, it searches and picks a random book through Open Library's api. Once it finds the book, the program downloads the cover of it and adds the book's information to a csv file. The program then posts a tweet with the bookcover as an image and with the title and publisher of the textbook as the caption through the Twitter API.

Black Jack

A Python card game that my group partner and I made for our CS:2110 final project! The user can choose the amount of rounds and players, and then experience a game of blackjack as expected. While my partner developed the UI for the starting window and game, I focused on programming the logic for the game. Such as the card system, scoring, the dealer, and multiplayer.

Link

Haunted Forest

Worked with Lauren McCalmont to develop a VR game using Unity. We developed a game where a giant ghost spider chases the player. The goal of the game is to find the keys for your truck. Both the truck and the keys spawn in a random spot on the map, adding replay value to the game. The player can also use light sources as safe spots. When the spider gets in the radius of an area with light, the spider will respawn in a random spot on the map.

Video

Flyswatters

Worked with a group in my Digital Arts class to develop a interactive flyswatter game using Max MSP. We wrapped a flyswatter in christmas lights and used cv.jit to detect the location of the swatter. The project displays the laptop's webcam feed and display images of flies and butterflies over it. If the project detects the swatter over the fly, it adds a point. If its over a butterfly, it subtracts a point. The player has a limited amount of time before the game ends.
