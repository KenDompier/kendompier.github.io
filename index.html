Intro

I excel in bridging the gap between humans and computers through intuitive design, making technology accessible and interactions quick and easy.

Being amongst the first generation of kids to grow up in a post-computer and internet world, I experienced my childhood majorly spent using them. Being surrounded by a digitizing world, I was soon to realize my love for technology, design, and creativity. This is the reason I'm pursuing a degree in Informatics.

I have gained lots of experience during my time at the University of Iowa! I've learned many technical skills such as C#, Python, HTML, database management, and so much more. I've also been applying my knowledge to personal projects as well, which you can check out here!

I also like to dive into other hobbies and creative endeavors in my free time and through student organizations and working for the university. I am currently a Senior Contracts Producer for the Student Video Production on campus, where we've done contract work for the university and have volunteered to produce videos for Dance Marathon. I also do work as an AV Technician for the university! Through these experiences, I learned the importance communication with clients and organization when working with a team.

Projects

MiLibrary

MiLibrary is a program I developed using C# and WPF which acts as a journal for the user's media consumption. The user is able to log when they started the media, finished the media, and give it a rating. The program saves the inputted media when the user closes it and loads it when the program opens. The user can also compare the ratings between the media they've consumed and see how much of each media type exists saved in the data through pie charts that can be opened through the program.

Random YouTube Generator

I wrote this Python program to compensate for YouTube's lack of a random video feature. It generates a string of 3-5 random characters and searches for it with the YouTube API. When the program recieves the first result, it returns the link and opens it with the user's default search browser. The program also has the ability to search for video by length and before a certain year. I also gave the program a cute little face that reacts to errors!

Textbook Twitter Bot

My first Twitter bot using Python! From a list of a few different textbook publishers, it searches and picks a random book through Open Library's api. Once it finds the book, the program downloads the cover of it and adds the book's information to a csv file. The program then posts a tweet with the bookcover as an image and with the title and publisher of the textbook as the caption through the Twitter API.

Projects

GroceryGo

Designed a smartphone application with Noah Knosp and Guadalupe Camacho to help users compare prices between local grocery stores. We worked as a team to wireframe and develop a low-fidelity and high-fidelity prototype using HTML, Bootstrap, and ReactJSX. We then followed this by conducting usability testing with our target userbase and having them complete a NASA-TLX form to determine issues with our design as we further developed it.

Haunted Forest

Worked with Lauren McCalmont to develop a VR game using Unity. We developed a game where a giant ghost spider chases the player. The goal of the game is to find the keys for your truck. Both the truck and the keys spawn in a random spot on the map, adding replay value to the game. The player can also use light sources as safe spots. When the spider gets in the radius of an area with light, the spider will respawn in a random spot on the map.

Flyswatters

Worked with a group in my Digital Arts class to develop a interactive flyswatter game using Max MSP. We wrapped a flyswatter in christmas lights and used cv.jit to detect the location of the swatter. The project displays the laptop's webcam feed and display images of flies and butterflies over it. If the project detects the swatter over the fly, it adds a point. If its over a butterfly, it subtracts a point. The player has a limited amount of time before the game ends.

Black Jack

A Python card game that my group partner and I made for our CS:2110 final project. The user can choose the amount of rounds and players, and then experience a game of blackjack as expected. While my partner developed the UI for the starting window and game, I focused on programming the logic for the game. Such as the card system, scoring, the dealer, and multiplayer.
